## 本リポジトリについて
なかなか外に出ることが難しい昨今、自宅で道具なしのトレーニングを支援するアプリケーションが欲しいと思い、
自分のために作成した筋トレ支援アプリケーションのリポジトリとなります。

その日の体調、鍛えたい部位、使える時間などを入力すると、自動でトレーニングメニューを生成してくれます。
